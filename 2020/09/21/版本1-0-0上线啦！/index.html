版本1.0.0上线啦！

  经过作者大大两三天的不懈努力n，版本1.0.0，即最初的正式版上线了。此版本基于Hexo框架和Next主题，在Next主题上经过美化并且增加了许多的功能，下面就一起来看看这些功能吧🎉！

>
银河

背景音乐

  在原Next主题上增加了网易云音乐外部播放器，这里选用的是薛之谦的天外来物。在侧边栏就可以看到，并且可以控制进度，可以随意的暂停与开始。在此基础上背景音乐也实现了全局播放。

界面美化

  添加了全局背景，一张自认为还不错的图片。在主页以及各个页面的主体部分均有淡黄色背景衬托，方便文章的阅读。并且底部与顶部均有圆角的美化，是整体看起来更加的柔和。

菜单栏

  在菜单栏加入了首页，标签，分类，归档，站点地图和搜索选项，大家可以根据需要自行查看。

首页

  在首页展示了网站的基本样貌，以及所有的已发布的文章，每篇文章都有发表时间，分类（如果有的话），更新日期（如果更新过）和浏览量。如果文章较长，我会在文章中加入预览部分，即在首页中可以看到的部分。如果需要看到更多内容只需点击阅读全文按钮即可。

标签

  在这部分你可以点击你想要看到带有你想要看的标签的文章。点击标签即可出现所有带有此标签的文章。

分类

  在这部分你可以看到所有不同类别的文章，每一片文章只能属于一个分类，所以作者大大也就是我会尽量将每一篇文章放到适合它的那类。

归档

  在这部分你可以看到有史以来作者大大发布的全部文章，按照时间顺序排列，从早到晚。

站点地图

  因为站点地图同时涉及到搜索引擎的搜索爬虫，对于新手的作者大大来说任然有诸多不确定，所以在1.0.0正式版中站点地图仍然不能使用。不过作者大大可以告诉大家，我已经将我的博客添加到谷歌以及百度的搜索引擎中了，只不过仍然面临一些问题。作者大大会在后续的版本更新中完成这一功能。

搜索

  在这部分就像所有的搜索网站一样，你可以搜索你想要看到的内容，不过只能搜索内容。更高级的功能，例如添加搜索的时间范围。

评论功能

  在1.0.0正式版中本博客添加了评论功能，而且是不需要登陆即可留言的评论功能⭐️。评论区由valine驱动。

置顶功能

  所有的文章都有置顶功能。如果有必要的话，例如版本更新信息，精品文章，都会置顶的。

期望

  在未来的更新中，作者大大会添加更多有趣的功能。

可以确定要添加的功能：

· 打赏功能
· 直接引用YouTube视频（国内的朋友可能就看不到了，不过会有其他的方法让你们看到的）
· 直接引用BiliBili视频
· 文章的分享功能
· 文章中插入数学公式
· 插入表格
· 在线观影
· 背景音乐列表功能
· 文章完结标示
· 文章字数统计

未来所更新的文章方向

· 数学区：在这里会更新一些数学知识，从高中开始的知识内容。
· 计算机知识区：在这个区主要记录一些作者大大的学习中的随笔，希望可以对所有的新手有帮助。

最后作者大大希望这个博客网站可以和你们及我一起起成长，越变越好！
